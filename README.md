# Como Jogar:

•	Use as setas (esquerda, direita, para baixo) para mover as peças que caem para os lados e para baixo.
•	Pressione a tecla de seta para cima para girar as peças.
•	Tente completar linhas horizontais preenchendo-as com tetrominoes.
•	Limpe o maior número possível de linhas para marcar pontos.
•	O jogo termina quando as peças se empilham até o topo do campo de jogo.
